### PROJECT TITLE
Perturbation-based Simulation Modeling of Long-Latency Risk Contributions in Young-Onset Arthritis

### PROJECT DESCRIPTION
The project provides a quantitative simulation framework for analyzing risk factor contributions to the development of arthritis.
Key risk factors include BMI, age, smoking status, and genotypic frequency.
The simulation systematically perturbs the average BMI within age groups to identify a plausible age-dependent scaling effect of the impact of BMI on arthritis pathogenesis. Results are validated using the Global Burden of Disease Study 2023.
Input parameters have been searched on publicly available databases and sources for scientific information.
